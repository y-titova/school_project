# school_project

# Учимся считать ⭐

Учебный проект для ученика 1 класса.

## Описание
Обучающая игра для изучения сложения и вычитания:
- счёт яблок до 20
- примеры с числами до 100

За правильный ответ появляется звёздочка.

## Как запустить
1. Открыть файл `index.html` в браузере
2. Или добавить на главный экран Android-планшета

## Автор
Проект выполнен учеником 1 класса  
при поддержке взрослого
